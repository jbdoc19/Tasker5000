## Direct Answer

**Friction-Minimizing Chart Algorithm (FMCA)** ‚Äî a tiny, repeatable loop that prevents time blindness and stalls.

**Timers**

* Sprint = 25‚Äì30 min.
* Micro-timer = 5 min ‚Äúno progress‚Äù check.
* Swap trigger = 15 min on a single chart without progress.

**What counts as ‚Äúprogress‚Äù**: attestation signed, one section finished (HPI/PE/A&P), or a concrete decision logged (e.g., ‚ÄúNeed labs ‚Üí parked‚Äù).

**State Machine**

1. **Start** ‚Üí Pull 8 batched charts (attest-only first). Begin Sprint timer and Micro-timer.
2. **5-min no-progress?** Run **Micro-Unstick (60‚Äì90 sec)**:
   a) Define finish line in one sentence.
   b) Commit smallest step (dictate 2 sentences or tick 1 checklist item).
   c) Resume; reset Micro-timer.
3. **12-min still slow?** Run **Accelerator**:
   a) Use template/snippet, or copy forward problem list.
   b) If info missing, insert a TODO placeholder and proceed.
   c) Reset Micro-timer.
4. **15-min stuck?** **Swap-3 Protocol**:
   a) **Park** current chart: add a one-line blocker note and a 3-item next-step checklist.
   b) Pull **three attest-only** charts and finish all three back-to-back.
   c) Return to the parked chart with a fresh **10-min cap**.
5. **Still stuck after return?** **Escalate**: park it again, schedule a **20-min Deep Fix** block later (not now), and continue the batch.
6. **Sprint end**: log wins, sip water, **2-min Reset** (stand, breathe, quick stretch), preload next batch of 8.

**Hard rules**

* Max **two mixed sprints** in a row; then do one easy sprint (attest-only).
* **Same-day required** or **‚â•80 days old**: allow only one Swap-3 cycle before Escalate.
* Daily cap = **4 sprints**. If energy remains, do a 15‚Äì20 min Lane B nibble only.
